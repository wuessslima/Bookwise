# BookWise - Personal Reading Management Application

WDD 330 Final Project - BYU Idaho

## Overview
BookWise is a comprehensive web application designed for personal reading management, helping users organize their literary journey, discover new books, and track their reading progress effectively.

## Features
- Virtual Bookshelf System
- Reading Progress Tracker
- Intelligent Recommendation Engine
- Advanced Library Search & Filtering
- Reading Statistics & Insights
- Review, Rating & Annotation System

## Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
